# Search in a Binary Search Tree

**Difficulty**: <span style={{ color: '#5CB85C', backgroundColor: '#DFF0D8', padding: '3px 5px', borderRadius: '4px' }}>Easy</span>

**Link:** https://leetcode.com/problems/search-in-a-binary-search-tree


## Description 

You are given the `root` of a binary search tree (BST) and an integer `val`.

Find the node in the BST that the node's value equals `val` and return the subtree rooted with that node. If such a node does not exist, return `null`.

**Example 1:**

<img src="https://assets.leetcode.com/uploads/2021/01/12/tree1.jpg" alt="Tree 1" />


**Input:** `root = [4,2,7,1,3], val = 2` 
**Output:** `[2,1,3]` 

**Example 2:**

<img src="https://assets.leetcode.com/uploads/2021/01/12/tree2.jpg" alt="Tree 2" />

**Input:** `root = [4,2,7,1,3], val = 5`  
**Output:** `[]` 


## Solution
Solved this one after watching the NeetCode video on binary search trees in his *Algorithms and Data Stuctures for Beginners* course.

```python
class Solution:
    def searchBST(self, root: Optional[TreeNode], val: int) -> Optional[TreeNode]:
        if not root:
            return

        if val > root.val:
            return self.searchBST(root.right, val)
        elif val < root.val:
            return self.searchBST(root.left, val)
        else:
            return root
```
