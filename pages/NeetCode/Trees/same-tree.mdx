# Same Tree

**Difficulty**: <span style={{ color: '#5CB85C', backgroundColor: '#DFF0D8', padding: '3px 5px', borderRadius: '4px' }}>Easy</span>

**Link:** https://leetcode.com/problems/same-tree


## Description 

Given the roots of two binary trees `p` and `q`, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.

**Example 1:**

<img src="https://assets.leetcode.com/uploads/2020/12/20/ex1.jpg" alt="ex1" />


**Input:** `p = [1,2,3], q = [1,2,3]`  
**Output:** `true` 

**Example 2:**

<img src="https://assets.leetcode.com/uploads/2020/12/20/ex2.jpg" alt="ex2" />

**Input:** `p = [1,2], q = [1,null,2]`  
**Output:** `false` 

**Example 3:**

<img src="https://assets.leetcode.com/uploads/2020/12/20/ex3.jpg" alt="ex3" />

**Input:** `p = [1,2,1], q = [1,1,2]`  
**Output:** `false` 

## Solution

```python
class Solution:
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        if not p and not q:
            return True
        if not p or not q:
            return False
        if p.val != q.val:
            return False
        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
```
