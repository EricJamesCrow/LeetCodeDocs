# Maximum Depth of Binary Tree

**Difficulty**: <span style={{ color: '#5CB85C', backgroundColor: '#DFF0D8', padding: '3px 5px', borderRadius: '4px' }}>Easy</span>

**Link:** https://leetcode.com/problems/maximum-depth-of-binary-tree


## Description 

Given the `root`of a binary tree, return its maximum depth.

A binary tree's **maximum depth** is the number of nodes along the longest path from the root node down to the farthest leaf node.

**Example 1:**

<img src="https://assets.leetcode.com/uploads/2020/11/26/tmp-tree.jpg" alt="Tree" />


**Input:** `root = [3,9,20,null,null,15,7]`  
**Output:** `3` 

**Example 2:**

**Input:** `root = [1,null,2]`  
**Output:** `2` 


## Solution #1

This was my first attempt at this problem. I could not think of a clean solution at the time so I went back to the NeetCode video.

```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        root_left, root_right = 1, 1
        if root.left:
            root_left += self.maxDepth(root.left)
        if root.right:
            root_right += self.maxDepth(root.right)
        if not root.left and not root.right:
            return 1
        if root_left > root_right:
            return root_left
        else:
            return root_right
```


## Solution #2 

NeetCode's solution.

```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        
        return 1 + max(self.maxDepth(root.left), self.maxDepth(root.right))
```
